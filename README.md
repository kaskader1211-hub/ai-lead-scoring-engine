# ai-lead-scoring-engine
AI-powered lead scoring system that automatically evaluates and prioritizes sales leads using GPT-4o. Built with n8n, Typeform, Airtable, Monday.com and Slack.
Automatyczny system scoringu leadów oparty na AI, który ocenia i priorytetyzuje potencjalnych klientów używając GPT-4o. Zbudowany z n8n, Typeform, Airtable, Monday.com i Slack.
